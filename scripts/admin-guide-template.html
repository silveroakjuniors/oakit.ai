1. Dashboard Overview

1

2

3
Select class and sections
4

5
Save assignmentActivities auto-rotate through the pool in order

How Rotation Works

  • Activities cycle through the pool in the order you set
  • Each section gets the next activity in sequence on their scheduled day
  • The system tracks a "rotation cursor" per section
  • If a day is a holiday, the activity shifts to the next working day

Managing the Schedule

  • View Monthly Schedule: See which activity is assigned to which section on each day
  • Reschedule: Move an activity to a different day if needed
  • Pool Summary: View all pools with their current rotation position
  • Delete Activity: Remove from pool (shifts remaining activities up)

13. Smart Alerts & AI Usage

Smart Alerts

AI-computed intelligence alerts that highlight issues requiring your attention.

Alert Categories

Teachers Not Completing Plans

Teachers with broken streaks, unlogged days, or consistently late completions

Low Attendance Trends

Sections where 7-day rolling attendance drops below 70%

Curriculum Behind Schedule

Classes with coverage below 40% - falling behind on syllabus

Weak Subject Detection

Subjects where quiz average scores are below 50%

Teacher Performance Scores

Multi-factor scoring for each teacher based on:

  • Plan Compliance: % of days with plans marked as done
  • Attendance Marking: Timeliness and consistency
  • Homework Sent: Frequency of homework communication
  • Teaching Streak: Consecutive days of completion
  • AI Engagement: How often they use Oakie for teaching help
  • Observations: Number of student observations recorded
  • Parent Communication: Journey notes and messages sent

Teacher Performance Drill-Down

  • Click any teacher to see daily breakdown
  • View homework submissions by week
  • AI usage patterns (which questions they ask)
  • Engagement metrics over time

Oakie AI Usage

Monitor and manage your school's AI credit consumption.

1
Go to Oakie Usage from sidebar
2
View credit wallet balanceCurrent credits remaining
3
Check usage breakdownMonthly call count, cost per call, usage by endpoint

What You See

  • Wallet Balance: Current AI credits available
  • Lifetime Usage: Total credits consumed since school onboarding
  • Daily Chart: 30-day usage graph showing daily AI calls
  • By Endpoint: Which features consume the most credits (chat, plans, formatting)
  • Recharge History: Past credit purchases and dates
Monitor AI usage weekly. If credits are running low, teachers will be unable to use Oakie chat.
